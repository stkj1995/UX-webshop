## Installation
1. Install [`json-server`](https://github.com/typicode/json-server) globally to have it available for any other project in your computer: `npm i -g json-server`.
2. Run `json-server` to use the file `data/users.json` as a REST API: `json-server --watch data\users.json --host=127.0.0.1` (the backslash should be a slash for Linux and Mac).
3. Open the application in a browser.
